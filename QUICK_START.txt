Abre tu terminal en la carpeta del proyecto y ejecuta:

  1️⃣  npm install --legacy-peer-deps
  2️⃣  npm run build
  3️⃣  npm start

Luego abre: http://localhost:3000

═══════════════════════════════════════════════════════════════════

  ❌ Errores de dependencias:
     npm install --legacy-peer-deps

  ❌ Limpiar caché:
     rm -rf .next node_modules
     npm install --legacy-peer-deps

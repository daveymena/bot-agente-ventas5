# Landing Page Generator

Un generador profesional de landing pages con editor visual drag-and-drop.

## Características

- ✨ **Editor Visual**: Arrastra y suelta componentes
- 🎨 **Templates Profesionales**: Comienza con diseños predefinidos
- 📊 **Propiedades Editables**: Personaliza colores, fuentes y estilos
- 📥 **Exporta HTML**: Descarga tu landing page completamente funcional
- 💾 **Guarda Proyectos**: Almacena tu trabajo en la nube

## Tecnologías

- **Frontend**: React 19 + Next.js 16
- **Estilos**: Tailwind CSS
- **Lenguaje**: TypeScript
- **Drag & Drop**: DnD Kit

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Build Producción

```bash
npm run build
npm start
```

## Estructura del Proyecto

```
/landing-page-generator
├── app/              # Páginas de Next.js
├── components/       # Componentes React
├── contexts/         # Contextos (EditorContext)
├── types/            # Tipos TypeScript
├── public/           # Archivos estáticos
└── styles/           # Estilos globales
```

## Uso

1. **Inicio**: Ve a la página principal
2. **Selecciona Template o Inicia en Blanco**: Elige cómo empezar
3. **Añade Componentes**: Usa el panel izquierdo
4. **Personaliza**: Edita propiedades en el panel derecho
5. **Exporta**: Descarga tu landing page en HTML
